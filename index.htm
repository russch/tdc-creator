Tableau TDC File Creator

Who enjoys manually creating a TDC file? Not me!

Instructions:

1. Select a Data Source. Note that not all data sources respect TDC files so the list below isn't complete.

2. Select one or more Customizations.CTRL-Click (CMD-Click on Mac) to multi-select.

  • Customizations below may or may not apply to your Data Source. Contact your vendor for more info
  • Selected Customizations appear in the blue "You've Selected" list. Modify the value of each with the associated input box
  • If you choose Generic ODBC, you must manually enter a value for "driver". You can find this value by examining the datasource XML element in your twb or tds with a text editor. Look for child element named driver name

3. Click "Give Me" button. Wait for 4-5 seconds and myVeryOwnTDC.tdc will appear in your download list (based on browser you may not get a prompt asking if you want to download)

You've selected:
Nothing.
